## 📌 Overview

Empowering students with an intelligent digital library system that simplifies book discovery, borrowing, and management. This project integrates Natural Language Processing (NLP) techniques to recommend books based on user input, enhancing the overall reading and search experience.



## 🎯 Objective

Managing library resources manually can be time-consuming and inefficient. This project aims to:

* Digitize library operations
* Simplify book searching and borrowing
* Provide intelligent book recommendations using NLP
* Improve user experience through automation and personalization

## 📊 Dataset

**Source of Dataset:** Book dataset used for recommendation system

**Description:**
The dataset contains:

* Book Title
* Description
* Category
* Author

This data is processed using NLP techniques to recommend similar books.



## ⚙️ Installation

Steps to run the project:

```bash
git clone https://github.com/tejashree-pawar/nlp-mini-project.git
cd nlp-mini-project
```

### Install Backend Dependencies

```bash
cd backend
pip install -r requirements.txt
```

### Install Frontend Dependencies

```bash
cd ../frontend
npm install
```



## ▶️ Usage

How to run the project:

### Start Backend

```bash
cd backend
uvicorn app.main:app --reload
```

### Start Frontend

```bash
cd frontend
npm run dev
```

### Access the Application

Open in browser:
http://localhost:5173
