##  Overview
This project is a Smart Library Management System with an NLP-based Book Recommendation feature. We developed a website where students can log in, view books, and borrow them, while admins can manage the library. The frontend is built using React, TypeScript, and Tailwind CSS, and the backend is developed using Python with FastAPI. We used Firebase as the database to store user and book data. The main feature of our project is the NLP recommendation system, which uses TF-IDF and cosine similarity to suggest similar books to users based on book titles or descriptions.

##  Objective
The objective of this project is to:
- Develop an online library management system  
- Help users easily find and borrow books  
- Implement an NLP-based recommendation system to suggest similar books  
- Improve user experience in searching and discovering books

## Usage
### Step 1: Run Backend (FastAPI)

Open terminal and run:

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload

Backend will start at: http://127.0.0.1:8000

Step 2:Run Frontend (React)

Open a new terminal and run:

cd frontend
npm install
npm run dev

Frontend will start at: http://localhost:5173

Step 3: Use the Application
Open the frontend URL in your browser
Register or log in as a user
Browse available books
Borrow books from the system
Enter book title or description to get recommendations

The system will suggest similar books using NLP techniques like TF-IDF and cosine similarity.
